欢迎来到Roger的博客

本人出生在素有武术之乡之称的广东佛山，虽然无学得一招半式但是耳濡目染也感受到其中的精神，是一种执着，是对每一招每一式“不将就”的坚持，这种态度融入我的工作和生活中

现在主要是在捣腾PHP，但是陆陆续续进的坑还有很多。。。。。。

  • Vue.js
  • MySQL
  • Redis
  • MongoDB
  • Python
  • Linux
